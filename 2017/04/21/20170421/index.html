在笔记本上安装Ubuntu和CentOS的桌面应用

**家里的笔记本电脑操作系统换成Linux已经快一年了，感觉实在是太好用了，基本上各种软件都有解决方法，虽然不是很全，但是基本上可以满足日常的应用了，这里将整个安装所需要的软件包列一下，基本上实现一次安装所需要的全部软件。

Ubuntu16.04安装

直接使用下面的命令来安装吧，除了常用的开发工具，编译器，还会安装一个可以满足使用的Xfce4桌面系统，

apt-get install ant apache2 bzip2 chromium-browser chromium-browser-l10n evince firefox firefox-locale-zh-hans filezilla fonts-wqy-microhei fonts-wqy-zenhei gimp ghex gnome-mplayer gzip mplayer mousepad mpv openjdk-8-jdk openjdk-8-jre openssh-client openssh-server p7zip p7zip-full p7zip-rar rar rdesktop remmina remmina-plugin-rdp remmina-plugin-telepathy remmina-plugin-vnc rsync slim smplayer  thunar-archive-plugin unrar unzip vim virtualbox wireshark zip alsa-utils audacious build-essential esound-common evince filezilla gpicview liferea network-manager network-manager-gnome openssh-server sylpheed sylpheed-i18n thunar-archive-plugin vlc mysql-workbench xfce4-power-manager  xfwm4-themes owncloud-client-l10n wireless-tools file-roller  xubuntu-icon-theme   xfce4-terminal  alsa-source debconf-utils module-assistant xfce4 owncloud-client alsa-base linux-sound-base libapache2-mod-php7.0 make autoconf automake m4

CentOS7安装

CentOS的安装要麻烦点，因为需要安装扩展的源，这里启用了epel和nux源，基本上可以满足要求nux主要是使用里面的多媒体资源。yum-axelget则是yum的多线程下载扩展，可以使下载更快。至于软件包可以见说明。

yum install epel-release
rpm -Uvh http://li.nux.ro/download/nux/dextop/el7/x86_64/nux-dextop-release-0-5.el7.nux.noarch.rpm
yum install yum-axelget

yum install xfce4-session xfce4-session-engines xfce4-settings xfce4-terminal xfce4-panel xfce4-appfinder thunar-volman thunar-archive-plugin  xfce4-pulseaudio-plugin xfce4-power-manager xfconf xfdesktop xfwm4 wqy-microhei-fonts.noarch wqy-unibit-fonts.noarch wqy-zenhei-fonts.noarch cjkuni-ukai-fonts lightdm vim gvim firefox ibus-table-chinese-wubi-haifeng.noarch libpinyin.x86_64 gnome-software.x86_64 owncloud-client.x86_64 vinagre evince p7zip unar nfs-utils inkscape gimp okular thunderbird java-1.8.0-openjdk.x86_64 VirtualBox-5.1.x86_64 steam wine mplayer vlc im-chooser *alsa* rdesktop remmina unrar unzip wireshark audacious  make autoconf automake m4 ark bc
